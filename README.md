# Python 情感分析系统 python347

本项目是基于Python机器学习的文本情感分类系统，采用B/S架构，使用Django框架搭建，使用Mysql数据库进行数据存储。

## 技术栈

- 后端：Python
- 前端：HTML/CSS/JavaScript
- 框架：Django
- 数据库：Mysql

## 功能模块

### 登录模块

- 界面简洁，用户可快速登录
- 新用户可一键注册

### 后台管理

- 首页显示文本数量、分类类别等信息，便于管理

### 文本分类

- 输入文本，系统自动进行情感倾向判断，精准分类

### 文本管理

- 已分类文本列表，支持删除操作

### 用户信息管理

- 查看用户信息，支持修改或删除

## 系统角色

- 普通用户：登录、文本分类、文本管理
- 管理员：后台管理、用户信息管理

## 运行环境

- Python环境
- Mysql数据库

## 部署步骤

1. 安装Python环境
2. 安装Mysql数据库
3. 部署Django应用
4. 导入数据库文件
5. 启动应用
